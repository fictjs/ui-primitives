# AlertDialog

Alert dialog wrappers built on top of `Dialog` with modal semantics and `role="alertdialog"` defaults.

## Components

- `AlertDialogRoot`
- `AlertDialogTrigger`
- `AlertDialogPortal`
- `AlertDialogOverlay`
- `AlertDialogContent`
- `AlertDialogTitle`
- `AlertDialogDescription`
- `AlertDialogAction`
- `AlertDialogCancel`
